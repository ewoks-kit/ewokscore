ewokscore |version|
===================

*ewokscore* provides an API to define workflows and implement tasks in `ewoks <https://ewoks.readthedocs.io/>`_.

*ewokscore* has been developed by the `Software group <https://www.esrf.fr/Instrumentation/software>`_
of the `European Synchrotron <https://www.esrf.fr/>`_.
