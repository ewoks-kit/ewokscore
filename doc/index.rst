ewoskcore |release|
===================

ewoskcore provides an API for defining workflows in Ewoks.

ewoskcore has been developed by the `Software group <http://www.esrf.eu/Instrumentation/software>`_ of the `European Synchrotron <https://www.esrf.eu/>`_.
